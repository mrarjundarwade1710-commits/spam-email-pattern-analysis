# Spam Email Exploratory Data Analysis (EDA)

## Project Overview
This project performs exploratory data analysis on a spam email dataset to identify patterns and statistical characteristics of spam and non-spam emails.

The goal is to understand the structure of spam messages using data cleaning, feature engineering, statistical analysis, and visualization techniques.

## Technologies Used
Python
Pandas
NumPy
Matplotlib
Seaborn
SciPy

## Analysis Steps
1. Data loading and inspection
2. Data cleaning
3. Feature engineering
4. Statistical analysis
5. Data visualization
6. Correlation analysis

## Key Insights
Spam emails often contain more URLs and special characters.
Spam emails tend to be longer than non-spam emails.
Certain word patterns appear more frequently in spam emails.

## Project Structure
data – dataset  
notebooks – Jupyter notebook analysis  
visuals – generated plots  
src – reusable python scripts
